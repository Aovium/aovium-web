# aovium-web

The official website for **Aovium LLC** — a veteran-owned custom software and technology services company.

## Quick Start

Open `index.html` in a browser, or serve locally:

```bash
# Python
python3 -m http.server 8000

# Node
npx serve .
```

## Structure

```
├── index.html          # Single-page site
├── css/
│   └── style.css       # Design system & styles
├── js/
│   └── main.js         # Animations & interactions
└── assets/
    ├── clean_logo_flat.svg
    └── clean_logo_refined.svg
```

## Tech

Static HTML/CSS/JS — no build step, no dependencies. Fonts loaded from [Fontshare](https://www.fontshare.com/).
